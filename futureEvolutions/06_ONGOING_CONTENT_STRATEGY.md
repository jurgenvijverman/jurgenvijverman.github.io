## 6.4 Measurement & Analytics Framework

**Requirement source:** SEO PDF — Measurement Plan; GEO PDF — monitoring.

### GA4 Event Tracking (verify and expand)
Current events:
- `click_to_call` — phone link clicks
- `generate_lead` / `contact_form_submit` — form submissions

Add these events:
- `page_scroll` — 50% and 90% scroll depth on blog posts
- `outbound_click` — clicks to external links (GBP, social, directories)
- `file_download` — if any downloadable content is added
- `faq_interaction` — FAQ accordion opens (which questions are most popular)

### Search Console Monitoring
- Set up Google Search Console property for avyclima.be.
- Monitor weekly:
  - Impressions and clicks by page group (service hubs, city pages, blog posts)
  - Average position for target keywords
  - CTR by page
  - Coverage issues (errors, warnings, excluded pages)
  - Core Web Vitals status

### GBP Insights Monitoring
- Track monthly:
  - Total views (search + maps)
  - Phone calls from GBP
  - Direction requests
  - Website clicks from GBP
  - Photo views

### Monthly Reporting Checklist
- [ ] Total organic sessions (GA4)
- [ ] Top 10 landing pages by organic traffic
- [ ] Keyword positions for P1 targets (GSC)
- [ ] Lead count (form submissions + phone calls)
- [ ] GBP views, calls, direction requests
- [ ] New reviews count and average rating
- [ ] Core Web Vitals status (all green?)
- [ ] New backlinks acquired (GSC or Ahrefs/SE Ranking)
- [ ] Content published this month (blog posts, GBP posts)
