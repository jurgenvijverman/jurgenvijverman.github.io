## Image Optimization Notes

When adding images to the site:
1. Use WebP format where possible (with JPG fallback)
2. Add `loading="lazy"` to all below-fold images
3. Always include `width` and `height` attributes to prevent CLS
4. Write descriptive Dutch `alt` text for every image
5. Compress images before upload (target: <200KB per image)
6. Name files descriptively: `daikin-perfera-split-airco-aalst.webp` not `IMG_4523.jpg`
